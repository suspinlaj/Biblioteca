# 📕 **Gestión de Biblioteca: IES Rey Fernando IV**

Una aplicación de gestión bibliotecaria que transforma el aburrido registro de libros en una experiencia interactiva y visualmente muy cuidada, guiada en todo momento por la simpática bibliotecaria del centro.

## 📘 **Acerca de este proyecto**

Este proyecto fue desarrollado para la asignatura de **Desarrollo de Interfaces** en el segundo año del ciclo (DAM 2). El objetivo principal era construir una aplicación de escritorio/tablet funcional enfocada en la gestión de datos (registro y consulta), pero dándole una capa de personalización UI/UX muy superior a las aplicaciones de gestión tradicionales. Para ello, diseñé una interfaz inmersiva con temática de biblioteca clásica y un personaje interactivo que da *feedback* al usuario.

## 🔧 **Funcionalidades y Características**

- **Navegación Intuitiva:** Menú superior (Principal, Alta, Consulta) que permite transicionar de forma fluida entre las distintas vistas de la aplicación.
- **Asistente Virtual (UI/UX):** Integración de un personaje ilustrado cuyos "bocadillos" de texto cambian dinámicamente para guiar al usuario según la pantalla en la que se encuentre.
- **Módulo de Alta (Registro):** Formulario completo para registrar nuevos volúmenes, que incluye campos de texto (Título, Autor, Editorial) y un sistema de carga visual para añadir la imagen de la portada del libro.
- **Módulo de Consulta (Búsqueda):** Interfaz de filtrado mediante *Radio Buttons* para buscar libros por Autor o Editorial, mostrando los resultados estructurados (Autor/Editorial, Título y Portada).

## 💻 **Tecnologías**

- **Lenguaje**: C#
- **Framework**: .NET MAUI (Multi-platform App UI) / XAML
- **Enfoque destacado**: Maquetación adaptada a pantallas horizontales, validación de formularios, gestión de imágenes de usuario y *Data Binding*.

## ⚙️ **Estado del proyecto**

Finalizado. Proyecto educativo enfocado en el Desarrollo de Interfaces, con las vistas principales y la navegación totalmente maquetadas y operativas.
